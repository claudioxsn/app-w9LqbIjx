
## Sobre o Projeto

API de cadastro e movimentação de produtos.

## Versão do PHP

 PHP 7.4.3

## Versão do Laravel

Laravel Framework 8.83.4

## Instalação do Projeto

Após efetuar o clone do projeto, executar os seguintes comandos: 

-> composer install; 

-> composer update; 

-> php artisan migrate
